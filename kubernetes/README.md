# Kubernetes エコシステム ソースコードリーディング

Kubernetes は、コンテナオーケストレーションのデファクトスタンダードである。
コントロールプレーン（API サーバー、スケジューラ、コントローラマネージャ）とノードコンポーネント（kubelet、kube-proxy）が連携し、宣言的なインフラ管理を実現する。
本ディレクトリは、その主要コンポーネントを構成要素ごとに分けて読み解く。
対象バージョンは v1.36.2 にそろえる。

## 収録ドキュメント

| コンポーネント | 役割 | 言語 | ライセンス |
|---|---|---|---|
| [Kubernetes 本体](kubernetes/README.md) | コントロールプレーンとノードコンポーネント。API サーバー、スケジューラ、kubelet、kube-proxy 等 | Go | Apache-2.0 |

> 各コンポーネントのコード引用は、対応するリポジトリの `v1.36.2` タグに固定する。
> 引用部分の著作権は各プロジェクトの著作権者に帰属する（引用の方針はリポジトリルートの[引用とライセンス](../README.md#引用とライセンス)を参照）。
