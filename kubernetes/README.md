# Kubernetes エコシステム ソースコードリーディング

Kubernetes は、コンテナオーケストレーションのデファクトスタンダードである。
コントロールプレーン（API サーバー、スケジューラ、コントローラマネージャ）とノードコンポーネント（kubelet、kube-proxy）が連携し、宣言的なインフラ管理を実現する。
本ディレクトリは、その主要コンポーネントを構成要素ごとに分けて読み解く。
対象バージョンは v1.36.2 にそろえる。
